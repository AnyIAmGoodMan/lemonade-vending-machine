# Лимонадный автомат

Статус: Работающий прототип. Проект разработан в рамках Московской предпрофессиональной олимпиады школьников 2024/2025 учебного года.

Проект представляет собой программно-аппаратный комплекс для автоматического смешивания жидкостей по заданным рецептам. Устройство предназначено для круглосуточной продажи газированной воды и лимонадов в местах с большим потоком людей.

## Задача

Разработать устройство, позволяющее производить смешивание жидкостей с максимальной точностью и выдавать готовый напиток пользователю. Автомат должен обеспечивать смешивание по шести рецептам, автоматизированную выдачу заказов и устойчивую работу при взаимодействии с пользователем.

## Реализованные функции

- Приём заказа через подсистему приёма и выдачи заказов.
- Смешивание жидкостей по шести рецептам.
- Автоматизированная выдача готовых напитков.
- Остановка работы при изъятии стакана во время наполнения.
- Возврат к работе после возвращения стакана.
- Поочерёдная выдача стаканов.
- Заказы от 1 до 4 напитков.

## Рецепты

1. Газированная вода (50 мл.)
2. Мятный сироп (10 мл.)
3. Апельсиновый сок (40 мл.)
4. Лимонад Мятный (80 мл. газированной воды + 20 мл. мятного сиропа)
5. Лимонад Заводной апельсин (30 мл. газированной воды + 50 мл. апельсинового сока)
6. Лимонад Тройной (35 мл. газированной воды + 45 мл. апельсинового сока + 10 мл. мятного сиропа)

## Технические характеристики

- Три секции для жидкостей: газированная вода, мятный сироп, апельсиновый сок.
- Зона установки ёмкостей объёмом от 1 литра.
- Мерные стаканы объёмом 100 мл.
- Погрешность смешивания не более 1 мл.
- Автоматическое позиционирование диспенсера.
- Датчики наличия стаканов.
- Автоматическая выдача стаканов в зону приёма заказов.
- Габариты не превышают 1.5 метра по каждой оси.

## Структура репозитория

Папки расположены в соответствии с требованиями олимпиады:

- /Программный код - код для Arduino.
- /3D-модели - 3D-модели в форматах .stl.
- /Видео - ссылка на видео испытаний.
- /Чертежи - чертежи деталей.
- /Фотогрфии - фотографии устройства и его частей.

## Компоненты системы

- Микроконтроллер Arduino.
- Сервоприводы для позиционирования диспенсера.
- Насосы для подачи жидкостей.
- Датчики наличия стаканов.
- Кнопочная панель для ввода заказа.
- Кинематическая система перемещения стаканов.
- 3D-печатные детали корпуса и механизмов.
